# Deploy docs app

Here documented that application deploy process

:::tip

Its cover Cloudflare Pages (as kek.pages.dev) setup and becomes just example

:::

1. Go to [Cloudflare dashboard](https://dash.cloudflare.com/?to=/:account/pages) (register account if you haven't)
2. Pages > `Create a project` > `Connect to Git`
3. Connect GitHub account, select repository, select branch
4. Framework preset doesn't matter, go to `Build command` and `Build output` fields

- Build command: `yarn nx build docs`
- Build output directory: `apps/docs/build`

5. Add one environment variable for correct NodeJS version:

- `NODE_VERSION` - `17.9.1`
